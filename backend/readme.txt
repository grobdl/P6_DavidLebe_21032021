

Prerequis
npm doit être installé sur votre machine

Installation
Vous pouvez installer le serveur avec node server. Le serveur tournera sur localhost:3000.
Si le serveur tourne sur un autre port pour une autre raison, le port sera indiqué au démarrage.
Pour lancer le serveur taper: 
node server ou nodemon server

Modèle des requêtes: Mongoose
Contrôle du nom de l'unicité du nom d'utilisateur: mongoose-unique-validator
Cryptage données utilisateur: Bcrypt
Protection des routes sensibles: JSON Web Token
Traitement de l'envoi de fichiers: Multer, fs
Modules de sécurité utilisés: helmet, express-bouncer
